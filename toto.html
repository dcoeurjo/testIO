View DGtal OBJ in browser using three.js

  1. Generate OBJ files using Board3D (see dgtal_export_test.cpp for an example).
  2. Convert OBJ files into three.js format using convert_obj_three.py (can be found in this directory).
  3. Include the following scripts in the header.

    <script type="text/javascript" src="three.js"></script>
    <script type="text/javascript" src="jquery.min.js"></script>
    <script type="text/javascript" src="three_viewer.js"></script>

  4. Add a div for each viewer. Filename, origin and scale are customizable using attributes.
  5. Works in a Reveal.js presentation
  6. ...
  7. Profit

<div class="three_viewer" three_json="kspace_test.json"></div>

<div class="three_viewer" three_json="pointset_test.json" three_shadow="false" three_origin="5,5,5" three_scale=".2"></div>
